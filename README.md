>The Lox language implemented in Rust.

## Getting Started

To run the Lox REPL:

```bash
$ cargo run
> 1 + 1
Number(1.0)
Plus
Number(1.0)
```

To run a Lox program from a file:

```bash
$ cargo run main.lox
```
